# Hospitl Management
Hospital Management System
==========================

> A complete Hospital Management System designed to manage hospital operations efficiently.
> This system helps manage patients, doctors, appointments, medical records, billing, and other hospital activities in an organized way.

Features
--------

> Patient Management

> - Add new patients  
> - Update patient information  
> - Remove patient records  
> - View patient details  
> - Store medical history  

> Doctor Management

> - Add and manage doctors  
> - Store doctor information  
> - View available doctors  
> - Manage doctor departments and specialties  

> Appointment System

> - Book appointments  
> - View scheduled appointments  
> - Manage appointment status  
> - Connect patients with doctors  

> Medical Records

> - Maintain patient diagnosis records  
> - Store prescriptions  
> - Track treatment history  

> Billing System

> - Generate patient bills  
> - Manage payments  
> - View billing details  

Technologies Used
-----------------

> - Python  
> - Object-Oriented Programming (OOP)  
> - File Handling / Database Management  
> - Data Structures
